# JD-auto-review
=====================
进入京东评价页，自动生成评价，打星

Usage
-----
安装Userscript

### For Chrome

[Tampermonkey](https://chrome.google.com/webstore/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo)
或者直接拖拽到 Chrome Extensions 安装
